## Folder Structure
| Folder | Description |
| - | - |
| [Connectivity](connectivity)|Examples showing how to connect Emulate3D to external sources.|
| [Connectors](connectors)|Examples showing how to utilize the Emulate3D API to manipulate connectors.|
| [Custom Selection Behavior](custom_selection_behavior) | Example showing how to utilize the Emulate3D API to execute custom behavior based on what is selected in the scene. |
| [Exporting](exporting)|Examples showing how to export files using the Emulate3D API.|
| [IO Browser](io_browser)|Example showing how to force IO Browser server values and how to access all other IO Browser information using the Emulate3D API.|
| [Undo Manager](undo_manager)|Examples showing how to use the Undo Manager API.|
