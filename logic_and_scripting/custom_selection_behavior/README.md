## Description
This example showcases where and how to configure specific actions based on what is currently selected in the Emulate3D scene window.

Provided is an [example model](CustomBehaviorOnSelectionExample.demo3dx\Model.demo3dx) containing visuals that showcase the [CustomSelectionLogic](CustomBehaviorOnSelectionExample.demo3dx\scripts\CustomSelectionLogic\CustomSelectionLogic.cs)  script capabilities.

## Usage
- Select any visual in the scene.
- Oberve how the visual colors change & that statements are printed to the **Message Log** window based on what is selected.
